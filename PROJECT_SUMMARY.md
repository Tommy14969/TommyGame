## 🚀 如何使用

### 快速开始（3步）

1. **下载JSNES库**
```bash
curl -o js/jsnes.min.js https://cdn.jsdelivr.net/npm/jsnes@1.2.0/dist/jsnes.min.js
```

2. **启动本地服务器**
```bash
python -m http.server 8000
```

3. **打开浏览器**
```
http://localhost:8000
```

### 完整ROM集成

将ROM文件放入 `roms/` 目录，命名格式：`{ID}-{English_Title}.nes`

详细说明请查看 `SETUP.md`
