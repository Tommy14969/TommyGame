# NES ROM Files Directory

## ⚠️ 重要提示

此目录用于存放NES游戏ROM文件。**ROM文件受版权保护**，本仓库不包含任何ROM文件。

## 📁 ROM文件命名规范

请按照以下格式命名ROM文件：

```
{ID}-{English_Title}.nes
```

示例：
```
1-Super_Mario_Bros.nes
2-Legend_of_Zelda.nes
3-Contra.nes
4-Tank_Battle.nes
5-Pac-Man.nes
```

## 🔍 获取ROM的合法途径

### 1. 从正版卡带提取
如果您拥有正版NES卡带，可以使用以下设备提取ROM：
- USB CopyNES
- Kazzo
- INL Retro Programmer

### 2. 公共领域和授权ROM
一些游戏已经进入公共领域或获得授权：

- **Internet Archive**: https://archive.org/details/softwarelibrary_mame
- **Homebrew Games**: 独立开发的NES游戏
- **Demo ROMs**: 用于演示的ROM

### 3. 仅用于个人学习
本项目仅供学习和研究使用：
- 不得用于商业用途
- 请在24小时内删除未授权的ROM
- 支持正版，购买官方重制版

## ⚙️ 测试用ROM

如果您想测试模拟器功能但没有ROM，可以：

1. **使用测试ROM**: NESdev社区提供了一些用于测试的公共domain ROM
2. **自制游戏**: 搜索 "NES homebrew ROMs"
3. **Demo程序**: 使用简单的NES demo程序
