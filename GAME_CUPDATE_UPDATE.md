# 🎨 游戏封面图功能 - 更新说明

## ✨ 新增功能

已为所有44个游戏添加了复古风格的封面图预览！

### 更新内容

1. **游戏封面图生成**
   - 为44个游戏自动生成了复古NES风格封面图
   - 每张图片包含游戏主题色、中英文名称、NES标识
   - 图片尺寸: 400x300像素

2. **代码更新**
   - ✅ 更新CSS以支持图片显示
   - ✅ 修改游戏卡片渲染逻辑
   - ✅ 添加图片加载失败时的后备方案（显示原图标和色块）

3. **生成工具**
   - 创建了 `tools/generate_game_covers.py` 脚本
   - 可随时重新生成或自定义封面图样式

---

## 🎮 如何使用

### 查看效果

1. 双击运行 **`启动游戏.bat`**
2. 打开浏览器访问游戏站点
3. 现在每个游戏卡片都会显示精美的封面图！

### 自定义封面图

如果想更换某个游戏的封面图：

1. 准备一张400x300像素的图片
2. 命名格式: `{游戏ID}-{英文名(空格替换为下划线)}.png`
   - 例如: `1-Super_Mario_Bros.png`
3. 放入 `images/` 目录

### 重新生成所有封面

```bash
python tools/generate_game_covers.py
```

## 🎨 封面图设计特点

- **复古NES风格** - 经典红白机配色
- **渐变背景** - 使用游戏主题色
- **双层标题** - 中文主标题 + 英文副标题
- **游戏ID徽章** - 右上角显示游戏编号
- **NES水印** - 左上角品牌标识
- **像素图案** - 背景添加像素格子装饰
