## 📋 命名规范

所有封面图片必须按以下格式命名：

```
{游戏ID}-{英文名(空格替换为下划线)}.png

例如：
1-Super_Mario_Bros.png
2-The_Legend_of_Zelda.png
3-Contra.png
...
```

**注意：**
- ID必须是数字
- 英文名的空格改为下划线
- 使用.png格式
- 放入 `images/` 目录
